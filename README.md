# 🏥 NEXCARE AI
### AI-Powered Healthcare Assistant using Google Gemma 4

---

# Problem Statement

Healthcare information is often difficult for patients to understand. Prescriptions contain medical abbreviations, diagnostic reports are filled with technical terminology, and managing medicines, appointments, and health records becomes challenging—especially for elderly users and people with limited medical knowledge.

Many existing healthcare applications focus only on appointment booking or digital records but lack an intelligent assistant capable of explaining medical information in simple language.

**NEXCARE AI** solves this problem by providing an AI-powered healthcare assistant that helps users understand prescriptions, summarize medical reports, manage medications, and communicate healthcare information in an easy and accessible way.

---

# Project Description

NEXCARE AI is a modern AI-powered healthcare platform designed to simplify healthcare management through an intelligent virtual assistant powered by **Google Gemma 4**.

The platform provides an intuitive dashboard where users can:

- 💬 Chat with an AI healthcare assistant
- 💊 Analyze prescriptions
- 📄 Summarize medical reports
- 📅 Book doctor appointments
- ⏰ Manage medicine reminders
- 📚 Access trusted medical resources
- 📊 View healthcare analytics


The application features a premium UI inspired by Apple Health, ChatGPT, Material 3, and Notion AI, making healthcare technology accessible and easy to use for everyone.

The frontend is production-ready and designed for seamless integration with a FastAPI backend and Google Gemma 4 APIs.

## How Google AI Was Used

Google Gemma 4 is the intelligence behind NEXCARE AI
The frontend has been designed specifically to integrate with Gemma 4 APIs.

The AI assistant will be responsible for:

- Understanding prescriptions
- Extracting medicine information
- Summarizing medical reports
- Explaining complex medical terminology in simple language
- Providing multilingual healthcare guidance
- Answering healthcare-related questions
- Suggesting questions users can ask their doctors
- Generating easy-to-understand health summaries

The application architecture separates the frontend and backend, allowing FastAPI to communicate with Google Gemma 4 through secure API endpoints.

# Installation Steps

## 1. Clone the Repository

```bash
git clone https://github.com/akshaylekshman/buildathon_-final-project
```

## 2. Navigate to the Project

```bash
cd MediBridge-AI
```

## 3. Install Dependencies

```bash
npm install
```

or

```bash
yarn install
```

---

## 4. Start Development Server

```bash
npm run dev
```

---

## 5. Open in Browser

```
http://localhost:3000
```
